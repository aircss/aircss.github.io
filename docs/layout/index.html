Layout

The layout is the arrangement of visual elements on a page. There is three kinds of properties available to set the high-level page layout, the position of each element and the size of the elements. Every properties support media query breakpoints.

Coordinates
The coordinates sets where an element is positioned inside of a document or another element.
Display
The display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children.
Flexbox
Flexboxes are first-class citizens in aircss to build advanced layouts.
Position
The position sets how an element is positioned in a document. This module should be used in conjunction with the coordinates module.
Spacing
Spacing allows to set the outer (margin) and inner (padding) space of an element.
Vertical Align
This module allows to set vertical alignment of an inline or inline-block box.
Z-Index
The z-index property specifies the stack order of an element.
